## üöÄ Overview
**DeepDetect AI** is a state-of-the-art SaaS platform designed to combat the rising threat of synthetic media. Utilizing the power of **Google's Gemini 1.5 Flash Vision Model**, it provides instantaneous, high-accuracy analysis of images and text to determine authenticity.

## ‚ú® Features
-   **üîç Multi-Modal Detection**: Analyzes Images, Text, and ID Documents.
-   **‚ö° Real-Time Analysis**: Get results in milliseconds with edge-optimized processing.
-   **üìä Advanced Forensics**: Detailed breakdown of Perplexity, Burstiness, and Error Level Analysis (ELA).
-   **üîê Bank-Grade Security**: Encrypted user sessions and secure file handling.
-   **üìà Visual Analytics**: Interactive charts and forensic reports (PDF Export capable).
-   **üíé Premium UI**: Glassmorphism design with `framer-motion` animations.

## ‚ö° Installation

### Prerequisites
-   Node.js (v18+)
-   MongoDB Instance
-   Google Gemini API Key

### Steps

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/your-username/deepdetect-ai.git
    cd deepdetect-ai
    ```

2.  **Server Setup**
    ```bash
    cd server
    npm install
    # Create .env file with GEMINI_API_KEY and MONGO_URI
    npm run start
    ```

3.  **Client Setup**
    ```bash
    cd client
    npm install
    npm run dev
    ```

4.  **Access**
    Open `http://localhost:5173` to view the application.
